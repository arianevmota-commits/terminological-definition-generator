# Terminological Definition Generator

This repository contains the ontology and the Python program developed during the PhD research entitled:

**The Semi-Automation of Terminological Definitions from Ontologies: Challenges and Perspectives**

## Description

This repository presents the computational resources developed to support the semi-automatic generation of terminological definitions in the Natural Sciences domain.

The ontology was developed in OWL using Protégé and models concepts, conceptual relations and terminological variants extracted from Brazilian Natural Sciences textbooks.

Based on this ontology, a Python program retrieves the formalized conceptual relations and organizes them according to predefined definitional models, supporting the drafting of terminological definitions.

## Repository structure

```
ontology/
    Ontologia_v1.owl

python/
    terminological_definition_generator.py
```
